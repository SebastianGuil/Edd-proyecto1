Este programa tiene como fin representar una red social en la que se muestran cada uno de los usuarios y sus relaciones de amistad. Los usuarios se representan como nodos de un grafo,
y las relaciones de amistad como aristas del mismo grafo. El propósito principal de este programa es decretar si existen grupos de amigos mediante componentes fuertemente conectados.
